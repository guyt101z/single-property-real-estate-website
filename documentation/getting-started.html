Yeogurt

The project has been scaffolded with a yeoman generator called yeogurt. You can learn more about the tool by viewing its Github documentation. The tool is NOT required to make any modifications to the project rather it can be used to further scaffold the project to fit your needs

Directory Structure

Inside the project you should see directory structure similar to the following :

root
- gulp
- src
-- _data
-- _html
-- _images
-- _layouts
-- _modules
-- _scripts
-- _styles
-- fonts
- prebuilt-templates

Directory structure follows Yeogurt standards.
Folders prefixed with an underscore (_) will have it removed when moved to build target (ex: src/_images -> build/images).
Folders NOT prefixed with underscore (_) will be copied to build target 1 to 1 (ex: src/fonts -> build/fonts)

Preview

To preview a project you need to have project dependencies installed as well as the gulp cli.

Install Gulp CLI :

npm install gulp-cli -g

Install project dependencies :

npm install

Serve html files for preview:

gulp serve

Create a build :

gulp --production

Preview Precompiled

You do not need to have project dependencies installed to preview precompiled HTML templates in the ./templates directory.

Frontend Dependencies

  • Bootstrap 3 Glyphicons here

To install those dependencies run the following command in the root of the project:

Templating Engine

Project HTML templates are built with nunjucks templating engine.

HTML Sections

Project is composed of multiple HTML sections located in the src/_html directory. They are referenced in the src/index.nunjucks file.

Customizing Styles

Project theme has been created with a CSS preprocessor called SASS. You may change the theme of the project by updating the SASS $primary variable in src/_styles/main.scss.

Customizing Javascript

Main javascript code is in src/_scripts/main.js. Frontend dependencies are referenced in the 

package.json
pre>. Project uses browserify to bundle dependencies. Javascript libraries that do not follow CommonJS standards need to be shimmed by browserify-shim.

package.json 
	
    "browserify": {
    "transform": [
      "browserify-shim"
    ]
  },
  "browser": {
    "jQuery": "./node_modules/jquery/dist/jquery.js",
    "bootstrap": "./node_modules/bootstrap/dist/js/bootstrap.js",
    "jqueryValidate": "./node_modules/jquery-validation/dist/jquery.validate.js"
  },
  "browserify-shim": {
    "jQuery": "global:jQuery",
    "bootstrap": {
      "depends": [
        "jQuery:jQuery",
        "tether:Tether"
      ]
    },
    "jqueryValidate": {
      "depends": [
        "jquery:jQuery"
      ]
    }
  }

Meta Tags

Meta tags and external dependencies are in src/_layouts/base.nunjucks.

Google Maps

Google maps is loaded and initiated in the base layout src/_layouts/base.nunjucks. Its API key needs to be replaced with your own API key.  The Google maps ID tags are in src/_html/map.html. 

Carousel

Carousel uses the Bootstrap carousel API's. Carousel images are located in src/_images/carousel directory and are referenced in src/_html/carousel.html.

Precompiled Templates

./templates directory contains precompiled HTML templates. Each template contains all the necessary javascript and stylesheet dependencies to get a website up and running.

Tutorials

Form Submission

After you have installed the necessary dependencies you may want to start modifying the templates to make it your own. Start by going to the ./src/_scripts/main.js directory and add an ajax call to save the the form submission information. 



$("form#submit-contact").validate({
  rules: {
    'name': "required",
    'phone': {
      required: true
    },
    'email': {
      required: true,
      email: true
    },
  },
  messages: {
    'name': "Please enter your name",
    'phone': {
      required: "Please enter a phone number"
    },
    'email': "Please enter a valid email address"
  },
  errorElement: "div",
  errorPlacement: function(error, element) {
    // Add the `help-block` class to the error element
    error.addClass("form-control-feedback");

    if (element.prop("type") === "checkbox") {
      error.insertAfter(element.parent("label"));
    } else {
      error.insertAfter(element);
    }
  },
  highlight: function(element, errorClass, validClass) {
    $(element).parents(".form-group").addClass("has-danger").removeClass("has-success");
  },
  unhighlight: function(element, errorClass, validClass) {
    $(element).parents(".form-group").addClass("has-success").removeClass("has-danger");
  },
   submitHandler: function(form) {
    /**
     *  If form validation is successful. Send me an email!
     */
   
    var datastring = $(form).serialize();
    $.ajax({
                url: "http://someAPI.com/save",
                type: "post",
                data: datastring,
                success: function(d) {
                    alert(d);
                }
            });
  }
});
